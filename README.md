Парсер имен
===========

Парсер имен в MongoDB (names, db.names) с `deti.mail.ru`

Парсятся первые пять страниц, для большего количества заменить в цикле число. Пример:

        "name" : "Аабхас",
        "sex" : "male",
        "alias" : "aabhas",
        "origin" : "индийское",
        "meaning" : "понимание"
